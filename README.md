# GYL-Server

Server of the localization system. See the [APIs](./doc/api.md) available.

## Installation

- Requirements

    - Maven 3.5.0

- Run locally

    ```bash
    $ mvn clean tomcat7:run
    ```

- Run unit tests

    ```bash
    $ mvn clean test
    ```
