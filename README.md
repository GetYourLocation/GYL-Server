# GYL-Server

Server of the localization system.

## Installation

1. Download [Maven](http://maven.apache.org/download.cgi).

2. Build and run:

    - foreground

        ```sh
        $ python server.py
        ```

    - background

        ```sh
        $ python deploy.py
        ```

3. The application should be available at http://localhost:8080
